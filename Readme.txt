Bonjour !

Ceci constitue notre rendu pour le TP2 de ISI. 
Notre binôme est composé de :

Alexandre LAVAL
Anais THORAL

Il s'agit d'un jeu Unity sur le modèle d'un endless shooter.
Le but du jeu est de détruire tous les asteroïdes avant qu'ils n'atteignent nos trois bases. 
Une fois les points de vie des nos bases réduits à 0 la partie est perdue.

L'objectif est donc de survivre le plus longtemps afin de marquer le plus de points possible.

Le meilleur score est sauvegardé.


Les contrôles sont :

Fleche du Haut / Fleche du bas pour ce déplacer
Barre d'espace pour tirer

Bon jeu !
